Install

기본 환경 설정

Environment Description Version
WINDOWS 운영체제    (이 프로그램은 Windows 환경에서만 작동합니다!) Windows 7 이상 (32bit/64bit)
GIT BASH Simulator를 다운로드하기 위한 프로그램 VERSION 2.13.0 이상
Visual Studio Simulator를 실행하기 위한 프로그램 VERSION Visual Studio12 이상

1. Gihub 계정 생성

https://www.github.com/ 에서 자신의 계정을 만듭니다.

2. Fork 하기

자신의 계정이 생성되었을 경우, OSS12 옆의 링크로 이동한후, Fork 버튼을 클릭합니다. (Warning! DOWNLOADZIP 을 클릭할 경우 프로그램이 실행되지 않습니다.)

3 .Git Bash 실행

GIT BASH 를 실행합니다.

4. Github 계정입력

git config --global user.name ○○○
git config --global user.email ○○○
명령을 입력한 후, 실행합니다.

5. Repository 생성

GIT BASH 를 실행합니다.

6. Clone 하기

git config --global user.name ○○○
git config --global user.email ○○○
명령을 입력한 후, 실행합니다.
