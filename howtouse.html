Install

기본 환경 설정

Environment Description Version
WINDOWS 운영체제    (이 프로그램은 Windows 환경에서만 작동합니다!) Windows 7 이상 (32bit/64bit)
GIT BASH Simulator를 다운로드하기 위한 프로그램 VERSION 2.13.0 이상
Visual Studio Simulator를 실행하기 위한 프로그램 VERSION Visual Studio12 이상

1. Gihub 계정 생성

https://www.github.com/ 에서 자신의 계정을 만듭니다.

2. Fork 하기

자신의 계정이 생성되었을 경우, OSS12 옆의 링크로 이동한후, Fork 버튼을 클릭합니다. (Warning! DOWNLOADZIP 을 통해 파일들을 Clone 할 경우 프로그램이 실행되지 않습니다.)

3 .Git Bash 실행

GIT BASH 를 실행합니다.

4. Github 계정입력

git config --global user.name ○○○
git config --global user.email ○○○
명령을 입력한 후, enter 를 칩니다.

5. Clone 하기

git clone https://github.com/okjcd123/OSS12 를 입력한 후 clone 을 실시합니다.

6. Visual Studio 실행하기

.. /OSS12/SolarSystem12/freeglutProject3D.sln 을 더블클릭한 후 Visual Studio 를 실행합니다.

7. Solar Sysytem Simulator 실행하기

Ctrl + F5 키를 눌러 컴파일을 실시합니다.

8. RUNNING

b

노란색 궤도선이 나타납니다. 다시 궤도를 제거하고 싶을시 같은 키를 다시 한번 누르시면 됩니다.

UP

현재 보이는 시야를 수직 위로 이동 시킵니다. 마우스 드래그나 다른 환경요인으로 인한 시야의 제일 마지막 위치를 기준으로 이동이 됩니다.

DOWN

현재 보이는 시야를 수직 아래로 이동 시킵니다. 마우스 드래그나 다른 환경요인으로 인한 시야의 제일 마지막 위치를 기준으로 이동이 됩니다.

LEFT

현재 보이는 시야를 왼쪽으로 이동 시킵니다. 마우스 드래그나 다른 환경요인으로 인한 시야의 제일 마지막 위치를 기준으로 이동이 됩니다.

RIGHT

현재 보이는 시야를 왼쪽으로 이동 시킵니다. 마우스 드래그나 다른 환경요인으로 인한 시야의 제일 마지막 위치를 기준으로 이동이 됩니다.

w

현재 보이는 시야를 앞으로 이동 시킵니다. 마우스 드래그나 다른 환경요인으로 인한 시야의 제일 마지막 위치를 기준으로 이동이 됩니다.

s

현재 보이는 시야를 뒤로 이동 시킵니다. 마우스 드래그나 다른 환경요인으로 인한 시야의 제일 마지막 위치를 기준으로 이동이 됩니다.

q

Manual 모드에서 사용자가 직접 행성계 자전 및 회전시킵니다. 1회 클릭시 대략 1~3일 기준으로 공전합니다.

r (소문자)

현재 시야에서 바닥을 기준으로 시계방향으로 원 회전을 합니다.

R (대문자)

현재 시야에서 바닥을 기준으로 반시계방향으로 원 회전을 합니다.

Using Mouse

Automatic Mode

마우스 오른쪽 버튼을 클릭한 후, 'Automatically'를 클릭할 경우 자동적으로 행성들이 공전 및 자전을 실시합니다.
다시 'Manually'를 클릭할 경우 'q' 키를 사용하여 수동으로 행성들을 움직일 수 있습니다.

Drag

출력 화면을 클릭한 후, 자신이 원하는 방향으로 드래그할 경우 시선이 드래그한 방면으로 바뀝니다.

Running Program

위의 사용법을 참고하여 프로그램을 활용하실 수 있으며, 추가적으로 Console 창에서 태양계의 각 행성별 각도에 따른 지구 날짜를 확인할 수 있습니다. 시작일은 태양계의 9개 행성이 일직선상으로 정렬하였던 1982년 3월 10일 기준으로 출력됩니다.
